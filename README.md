# 📊 Bangalore Tech Salary Decoder

A complete data analytics project built using **Python, Pandas, and NumPy** to analyze salary trends among Bengaluru technology professionals.

## 📌 Project Overview

This project simulates a real-world HR Analytics task where a messy salary dataset is cleaned, transformed, and analyzed to generate meaningful business insights.

The workflow includes:

- Data Inspection
- Data Cleaning
- Data Standardization
- Exploratory Data Analysis
- Business Analytics
- Insight Generation
- Reporting

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Google Colab / Jupyter Notebook

---

## 📂 Dataset Features

The dataset contains information such as:

- Employee ID
- Role
- Years of Experience
- Current Salary (CTC)
- Previous Salary
- Company
- Company Type
- Skills
- Education Tier
- Work Mode
- Location
- Joining Year

---

## 🧹 Data Cleaning Performed

- Renamed columns using snake_case
- Standardized job roles
- Converted salary values into numeric LPA format
- Standardized company types
- Standardized education tiers
- Standardized work modes
- Handled missing values
- Removed duplicate records

---

## 📈 Business Involved tasks

### 1. Salary Distribution by Role
Calculated:
- Mean Salary
- Median Salary
- Minimum Salary
- Maximum Salary

---

### 2. Experience Curve
Analyzed salary growth of SDE Backend professionals across experience levels.

---

### 3. Skill Premium Analysis
Compared salary differences for professionals with:

- AWS
- Machine Learning
- System Design
- Kubernetes

---

### 4. Company Premium
Compared salaries across:

- Unicorn
- MNC
- Mid-size
- Early-stage companies

---

### 5. Underpaid Employee Detection
Identified employees earning below the median salary of similar peers.

---

## 📊 Key Insights

- Product Managers recorded the highest median salary.
- Machine Learning showed the highest salary premium among the selected skills.
- Unicorn companies offered significantly higher salaries than MNCs for SDE Backend professionals.
- Experience strongly influenced salary growth.
